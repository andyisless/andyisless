GitHub 几点研究- 自留地

新研究出来得几点

1.404页面？

出现一个礼拜的404页面，会在setting中提醒为什么会404
出现问题：Your site is having problems building: The tag fancybox on line 77 in themes/landscape/README.md is not a recognized Liquid tag.
解决方案：删掉readme.md 即可！

2.为什么上传的东西在博客不更新？

因为这需要命令行来更新，需要hexo来发布
出错原因：一直用herminal终端，配合commit来发布到github的，结果没有经过hexo 所以没效果！
解决方案：用hexo工具！ 三步曲

hexo clean
hexo generate
(若要本地预览就先执行 hexo server)
hexo deploy

3.如何更改分类？

的确在G的时候需要设定好
另外一种方案？
解决方案：利用hexo 命令行的方式，添加tags分类
详情：https://www.cnblogs.com/hankleo/p/11606224.html

4.尝试修改之前的文章

markdown 写文章，要在标签之后，加入空格！
否则MD标签不生效
比如：
1.##这样不生效
2.## 加上空格最好！

5.如何设置repo名字，秒生效？

解决方案：rename为：repo名字.github.com

6.可以修改博客的发布时间！
